## üìö Course Overview

This course introduces core and advanced concepts in **Deep Learning**, combining theoretical explanations with practical implementations.  
Each topic is provided as a **Jupyter Notebook** that students can:

- Run cell by cell
- Modify and experiment with
- Use as a reference for assignments and projects

The notebooks include both **framework-independent implementations** and **Keras-based implementations** to help students understand concepts at different abstraction levels.
